# for
对于重复性任务，`for`会非常适合这种任务；与`while`不同，`for`允许执行一定次数的代码而不是一直执行。

```c

for(/*初始化表达式*/; /*条件表达式*/; /*更新表达式*/) {
    // 循环执行的代码
}

```

## 代码示例

假设我们想要打印1到10的数字，我们可以这么写。

```c
#include <stdio.h>

int main() {
    for (int i = 1; i <= 10; i = i + 1) {
        printf("%d\n", i);
    }
}

```

- `int i = 1`是初始化表达式，它是用来定义循环变量的初始值的。我们这里的循环变量就是`i`，它的值后续会在循环过程中被不断更新。
- `i <= 10`就是条件表达式，条件为真就会继续执行代码，条件为假就会跳出循环。
- `i = i + 1`就是更新表达式，在执行完代码之后，更新表达式会更新循环变量的值，以防止出现无限循环。

在此代码中，我们的初始表达式定义了变量`i`，随后我们的条件表达式进行判断`i`是否小于等于10，如果条件判断为真则执行`printf`打印变量`i`，随后，在执行完代码后会执行更新表达式，这会为变量`i`增加1；如此循环下去直到`i`大于11。

更新表达式可以被写成更加简洁的形式，比如`i = i + 1`可以简写成`i++`；`i = i - 1`可以简写成`i--`。

## break & continue
在循环中，我们会更常用到这两个关键词。

### break

`break`的作用是跳出循环，通常用于提前结束循环；举个例子，假设我们想攒2000块，每天攒100块。

```c
#include <stdio.h>

int main() {
    int money = 0; // 存款

    for (int i = 1; i <= 15; i++) {
        if (money >= 2000) {
            printf("攒够2000了！\n", i, money);
            break;
        }

        money = money + 100;
    }
}

```
在上面的代码示例中我们在`i`达到16后将不会继续攒钱，此时的存款为1500；
如果我们在开始时的存款为1900，那么在`i`达到2000后将会执行`if`内的代码并`break`，此时`for`循环会因为`break`而停止执行，即使`i`小于15。

### continue

`continue`的作用会跳过本次循环中后面剩下的代码，直接进入下一次循环；比如，如果我们只想打印出1到10之间的奇数，遇到偶数就就跳过。

```c

#include <stdio.h>

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) {
            continue; //如果 i 是偶数，跳过当前循环
        }

        printf("奇数:%d\n", i);
    }
}

```
在上面的代码示例中，每次会首先判断`i`是否为偶数，如果是偶数，那么会跳过这次循环，剩下的代码也不会被执行；如果是奇数，那么下面的`printf`打印将会被执行。

## for Vs. while
`for`与`while`都具有自己的擅长的领域，对于`for`，更适合于已知循环次数的情况下使用，比如我们要连续攒钱**15**天，在这种情况下便可以使用`for`。

```c

for (int i = 1; i <= 15; i++) {
    // 攒钱
}
```

而`while`循环更适合在循环次数不确定的情况下使用，因为不像`for`那样有明确的循环变量初始值和更新表达式，所以循环的次数不需要提前知道，一直循环到条件为假才停止。
这使得`while`循环在很多情况下会更加灵活，比如，让用户每次输入一个任意整数，直到输入数字0时退出程序。在这种情况下，我们无法提前直到用户会输入多少次整数，也就不知道要循环多少次，所以更适合`while`循环。

```c
#include <stdio.h>

int main() {

    int number;
    printf("请输入一个整数（如输入0则退出程序)：\n");
    scanf("%d", &number);

    while (number != 0) {
        printf("你输入的数字是：%d\n", number);
        scanf("%d", &number);
    }
}

```
