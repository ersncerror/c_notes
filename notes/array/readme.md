# 数组
在面对需要储存大量变量时需要定义多个变量，在很多情况下这并不理想且代码也会看着很石山，数组的作用便是提供一种将多个元素存放到一起的功能。

### 定义数组

```c

int arrayName[3] = {90};
// 数据类型 数组名称[数组大小] = {初始化值};
```

## 基础知识
在一个数组中，只能存放该数组指定的数据类型。例如，我们使用`int`来作为此数组可以存放的数据类型，因此在此数组中只能存放整数类型。
> C语言的任意数据类型都可以被数组存放，这取决于数组在定义时使用的数据类型。

在数组名称之后，是两个方括号（`[]`），里面可以定义数组大小，也就是该数组可以存放的最大元素数量。要注意的是：**在尝试施加超过数组允许的最大可存放的元素数量时，会出现警告或报错。**

数组大小后面，是两个花括号（`{}`），里面用于存放元素；在这里面，每个元素需要用逗号隔开。

我们可以在定义时就对该数组中的元素进行初始化，剩下未被赋值的元素会默认被赋为类型对应的零值。

```c

int scores[3] = {};
// 在初始化时没有任何元素被赋值，因此其数组中三个元素都会变为类型对应的零值
// 由于我们使用int来作为此数组的数据类型，因此数组中三个元素的值都为0。
// int scores[3] = {0, 0, 0};

int scores[3] = {12, 23};
// 在此数组中，因为前面的两个元素被赋值，所以不会被默认赋值为类型对应的零值。
// 剩下未被赋值的元素将会变为类型对应的零值，因此第三个元素的值会是0。
// int scores[3] = {12, 23, 0};
```

定义数组时也可以不提供数组大小，此时会根据初始化时数组内拥有的元素数量来决定该数组的大小。

```c

float scores[] = {82.5, 32.5, 54, 21};
// 方括号里没有定义数组大小，因此根据数组内拥有的元素数量来决定该数组的大小。
// 数组内有4个元素，那么数组的大小便会是4。
```

数组是有序排列的，这与排队相似，可以到队伍最前面找到第一位元素，也可以到队伍的最后面找到第四位元素。

每一个数组内的元素都有自己的索引值，索引值用来表示元素在数组内的位置。数组的索引值从0开始，从左往右依次递增。比如，有一个数组，大小为3，依次对应的索引值为：0，1，2。

通过索引可以获取数组中元素的值，要获取数组中第二个元素的值，可以这么写。

```c

float scores[3] = {82.5, 32.5, 54};

printf("%.1f\n", scores[1]) // 32.5
// 由于数组的索引从0开始，因此第二位元素的索引应为 1。
```

通过索引还能修改数组中元素的值，只需要在获取元素值的语句后面再加上赋值操作，这和给普通变量赋值的方法是一样的。

```c
float scores[3] = {82.5, 32.5, 54};

scores[1] = 25.1; // 使数组中第二个元素被赋值为25.1。

printf("%.1f\n", scores[1]) // 25.1
```

有时可能需要对数组内的元素逐一操作，这时可以使用`for`来做到。
假设我们要对数组中值低于5的元素进行打印索引和值。

```c

#include <stdio.h>

int main() {
    int numbers[10] = {10, 6, 4, 8, 2, 7, 12, 32, 9, 14};

    for (int i = 0; i < 10; i++) {
        if (numbers[i] < 5) {
            printf("Index: %d ", i);
            printf("Value: %d\n", numbers[i]);
        }
    }

    // 此处会进行for循环，从0到9依次检查数组中索引对应的元素值是否低于5
    // 如果低于5那么将会打印其索引和值。
}
```

假设我们现在希望在检查到元素值低于5时，那么将该元素的值加5。

```c

#include <stdio.h>

int main() {
    int numbers[10] = {10, 6, 4, 8, 2, 7, 12, 32, 9, 14};

    for (int i = 0; i < 10; i++) {
        if (numbers[i] < 5) {
            numbers[i] = numbers[i] + 5
        }
    }

    // 此处会进行for循环，从0到9依次检查数组中索引对应的元素值是否低于5
    // 如果低于5那么将值加5。
}
```
